# Why Custom Software Pays

A simple but solid landing page I built to explain why custom software is a better investment than off‑the‑shelf tools.

No fluff. Just stats, a real ROI comparison, and a contact form that actually works.

## Live Demo
[https://vertexlab-dev.netlify.app/](https://vertexlab-dev.netlify.app/)

## What's inside
- Clean, dark layout with cyan accents (Vertex Labs style)
- Key stats from Forrester, McKinsey, Gartner
- ROI chart showing how custom software pays off over 3 years
- Contact form with Netlify backend (messages go straight to my email)
- Mobile responsive
